# What does this plugin do?
Triggers a map vote once all the waves are completed, once the votes are in, change the map immediately. I recommend changing the tf_mvm_victory_reset_time and/or tf_mvm_victory_disconnect_time to a time that is extremely high. I put mine on 200.   

REQUIREMENTS:  
[Heapons' Nativevotes](https://github.com/Heapons/sourcemod-nativevotes-updated)
